# iPrint Utility
This is a Windows software for the Portable Mini Thermal Printer commonly sold as GB01 which can be found all over Aliexpress and Amazon!

[More information, demo and FAQ can be found here.](https://mywk.net/software/iprint-utility)

[Download from GitHub](https://github.com/Mywk/iPrintUtility/releases)

##### Other compatible printers (untested): MX10XW001, XW002, XW003, XW004, XW005, XW006, XW007, XW008, XW009, JX001, JX002, JX003, JX004, JX005, JX006, M01, PR07, PR02, GB01, GB02, GB03, GB04, LY01, LY02, LY03, LY10, AI01, GT01
